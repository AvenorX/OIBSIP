# 🌐 Marvel Landing Page

## 📖 Overview

The Marvel Landing Page is a modern, responsive, and visually engaging website inspired by the Marvel universe. It is designed to showcase clean layouts, smooth navigation, and an attractive user interface while demonstrating fundamental front-end development skills.

## 📸 Preview

![Landing Page Preview].(screenshot1.png)

## ✨ Features

- Responsive layout for desktop and mobile
- Sticky navigation bar
- Hero section with call-to-action button
- About section
- Features/Services section
- Modern UI design
- Smooth scrolling navigation
- Footer with social links

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

WebDev-L1-LandingPage

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it using Live Server in VS Code.
