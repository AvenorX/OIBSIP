# 🌡️ Temperature Converter

## 📖 Overview

Temperature Converter is a responsive web application that converts temperature values between Celsius, Fahrenheit, and Kelvin. The application provides instant conversion with input validation and an easy-to-use interface.

![Temperature Converter Preview](screenshot.png)

## ✨ Features

- Celsius Conversion
- Fahrenheit Conversion
- Kelvin Conversion
- Input Validation
- Error Handling
- Responsive Design
- Clean User Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
